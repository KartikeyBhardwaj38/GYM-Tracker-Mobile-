# IronLog — Advanced Gym Tracker

A professional mobile-first Progressive Web App (PWA) for tracking workouts, posture correction, abs, and knock knee exercises.

## Features
- 📅 Full calendar view with workout history
- 🤖 Smart Coach — AI-powered personalized suggestions
- 📈 Auto-upgrade system — automatically improves your plan weekly
- ✅ Set-by-set workout logger with rest timer
- 🏆 Personal Records tracker
- ⚖️ Weight log with chart
- 💪 Create custom plans by focus area
- 📱 PWA — install on phone like an app
- 💾 Offline support — works without internet

## How to Use
1. Open `index.html` in a browser
2. Tap the + icon in Plans to create your custom plan
3. Workout daily — tap exercises to expand and log sets
4. Smart Coach will suggest upgrades based on your performance

## Install on Phone (PWA)
### iPhone:
1. Open in Safari
2. Tap Share → "Add to Home Screen"

### Android:
1. Open in Chrome
2. Tap Menu → "Install App" or "Add to Home Screen"

## GitHub Pages Deployment
1. Create a new GitHub repo
2. Upload all files (index.html, sw.js, README.md)
3. Go to Settings → Pages → Source: main branch
4. Your app will be live at: `https://yourusername.github.io/gymtracker`

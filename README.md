# FitForge — Smart Gym Tracker

> A fully offline-capable Progressive Web App (PWA) for tracking workouts, fixing knock knees, improving posture, building abs, and tracking progress.

## 🔥 Features

- **Custom Logo** — branded FitForge identity
- **5 Pages** — Home, Workout, Calendar, Plans, Progress
- **70+ Exercises** — with full instructions, form tips, muscles worked
- **Smart Coach** — AI-powered personalized suggestions
- **Auto-Upgrade** — weekly plan progression system
- **Rest Timer** — animated countdown after each set
- **Progress Bar** — tracks completion during workout
- **Weight Chart** — visual weight trend (last 8 entries)
- **Body Measurements** — waist, chest, hips tracker
- **Personal Records** — auto-detected PRs with podium ranking
- **Calendar View** — full monthly history with completion rate
- **PWA** — install on phone, works fully offline

## 🚀 Deploy to GitHub Pages (Live Website)

1. Create a new GitHub repository (e.g. `fitforge`)
2. Upload `GymTracker.html`, `sw.js`, and `README.md`
3. Go to **Settings → Pages → Source: Deploy from branch → main → / (root)**
4. Your app will be live at: `https://YOUR_USERNAME.github.io/fitforge/GymTracker.html`
5. **Install on phone:** Open the URL in Safari (iPhone) or Chrome (Android) → "Add to Home Screen"

## 📱 Install as App

### iPhone (Safari):
1. Open your GitHub Pages URL in Safari
2. Tap **Share** → **Add to Home Screen**
3. Tap **Add** — it installs like a real app

### Android (Chrome):
1. Open the URL in Chrome
2. Tap the **3-dot menu** → **Install App** or **Add to Home Screen**

## 💾 Data Storage
All data is saved locally on your device using `localStorage`. Nothing is sent to any server.

Pure HTML + CSS + JavaScript — no frameworks, no dependencies, zero build process.
